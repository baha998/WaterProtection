# Developer Guide — WaterProtection

## Requirements
- Node.js 18+
- npm 8+
- Git

## Setup
```
git clone https://github.com/your-org/waterprotection.git

cd waterprotection
npm install
npm start
```

## Project Structure
- `/src` — main code  
- `/public` — static assets  
- `/docs` — documentation  
- `/components` — UI components  

## How to Build
```
npm run build
```

## How to Contribute
See `CONTRIBUTING.md`

## Coding Style
- Keep components small  
- Avoid unnecessary libraries  
- Prefer pure functions  
- Write clear commit messages
